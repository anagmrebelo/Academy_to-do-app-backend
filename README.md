# To Do App - Full stack project

App features:

-   Select user from list
-   View tasks by user
-   Edit a task
-   Delete a task
-   Add a new task by clicking on + buttom or pressing enter
-   Complete a task
-   Filter tasks by complete status
-   Sort tasks by due date
-   Persist tasks and user options in database
